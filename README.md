
# Python Quiz Game

## Description
A multiple-choice quiz game implemented in Python. The game asks questions in random order, validates user input, provides immediate feedback, and displays the final score with a percentage.  

My contributions:
- Designed the question structure and scoring system
- Implemented randomized question order and input validation
- Displayed feedback for correct and incorrect answers

## Features
- Multiple-choice questions
- Randomized question order
- Immediate feedback for each answer
- Final score and percentage displayed at the end

## Technologies Used
- Python

## What I Learned
- Handling user input and validation
- Working with lists and dictionaries
- Implementing randomized behavior using `random.shuffle()`
